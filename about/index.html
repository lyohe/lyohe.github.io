About Me

こんにちは。私は TypeScript と Go が好きなプログラマーです。現在はソフトウェア開発者（兼・経理）として働いています。

元々はさまざまな企業で10年ほど財務経理関連の業務に携わっていましたが、仕事の一環で社内向け会計データ処理基盤の開発にプロダクトマネージャーとして携わったことをきっかけにプログラミングに興味を持ち、2020年4月からプログラマーとして働きはじめました。

GitHub アカウントは lyohe です。これは私の本名が「りょうへい（Ryohei）」なのですが、以前ポルトガル出身の同僚に巻き舌で「お前の名前『ｯﾙﾘｮﾍ』っていうのか。すごく呼びづらいな！（意訳）」と言われ R じゃなくて L なのかなと思ってつけました。なお、その同僚は発音を諦めて「ﾘﾖ」と呼んでいました。

Twitter(X) アカウントは @rtsudal です。Twitter(X) ではプログラミング以外の話をすることも多いのですが、気が向いたらフォローしてください。

このブログには主に会計とプログラミングの話を書こうと考えています。学んだことや新たに知ったことを整理して書くことは、必ず自分自身の理解の促進に繋がると考えています。つまり自分のために書いているのですが、もちろん誰かの役に立つことがあれば嬉しいです。
