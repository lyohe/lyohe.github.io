About Me

こんにちは。私は TypeScript と Go が好きなプログラマー兼・経理として働いています。

元々はさまざまな企業で10年ほど財務経理関連の業務に携わっていましたが、仕事の一環で社内向け会計データ処理基盤の開発にプロダクトマネージャーとして携わったことをきっかけにプログラミングに興味を持ち、2020年からプログラマーとして働きはじめました。

GitHub アカウントは lyohe です。これは私の本名が「りょうへい（Ryohei）」なのですが、以前ポルトガル出身の同僚に巻き舌で「お前の名前『ｯﾙｲｮｴ』っていうのか。すごく呼びづらいな！（意訳）」と言われ R じゃなくて L なのかなと思ってつけました。なお、その同僚は発音を諦めて「ﾘ-ﾖ」と呼んでいました。
